# USD-VND Calculator

Tính toán tỷ giá USD - VND với tỉ lệ 80,000 VND = 10 USD.

## Tính năng
- Chuyển đổi VND ↔ USD
- Giao diện đẹp, responsive
- Tính toán nhanh, chính xác

## Cách sử dụng
1. Chọn đơn vị (VND hoặc USD)
2. Nhập số
3. Nhấn "Tính Toán"
